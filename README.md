# RetroVault

8Bit-Arcade is a retro NES ROM launcher built in Python. Play your favorite NES games with a sleek, classic interface.

## How to Use

### Run the Executable (Windows)
1. Download `nes.exe` from the releases.
2. Make sure the folder `NES Games/Games` contains your NES ROMs (`*.nes`).
3. you can add you favourite Nes Games ROMs inside NES Games/Games. 
4. Double-click `RetroVault.exe` to start.

### Run from Source
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/RetroVault.git
2. Install dependencies:
   ```bash
   pip install pygame
3. Run:
   ```bash
   python nes.py
